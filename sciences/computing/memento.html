Le memento du geek

SSH : La connexion sécurisée
Connexion sécurisée avec une machine distante ssh login@nom_de_domaine
Connexion sécurisée avec une machine distante et affichage du bureau ssh -X login@nom_de_domaine
Exécution d'une commande à distance ssh login@nom_de_domaine "commande"
Exécution d'un script local sur une machine distante ssh login@nom_de_domaine bash < script
Générer une paire de clé publique/privée ssh-keygen -t rsa
Envoyer la clé publique sur l'hôte distant ssh-copy-id -i id_rsa.pub login@nom_de_domaine


TAR : Création/Extraction d'une archive
Créer une archive à partir de fichiers (v :mode verbeux; f : pour spécifier les fichiers) tar -cvf nom_archive.tar fichier1 fichier2 ... fichierN
Créer une archive à partir d'un dossier tar -cvf nom_archive.tar dossier/
Créer une archive au format tar.gz à partir d'un dossier tar -czvf nom_archive.tar.gz dossier/
Créer une archive au format tar.bz2 à partir d'un dossier tar -cjvf nom_archive.tar.bz2 dossier/
Extraire une archive au format tar.gz tar -xzvf nom_archive.tar.gz
Extraire une archive au format tar.bz2 tar -xjvf nom_archive.tar.bz2
Lister le contenu d'une archive tar -tf nom_archive.tar


GZIP : Décompresser une archive
Décompresser un fichier .gz gzip -d fichier.gz


ALIAS : Créer un alias pour une commande
Créer un alias (dans le fichier "/etc/profile.d/my_rules.sh" par exemple) alias nom_alias='commande'


LN : Créer un lien
Créer un lien symbolique ln -s nom_du_fichier nom_du_lien


LS : Lister le contenu d'un dossier
Lister le contenu d'un dossier ls chemin_dossier
Lister le contenu d'un dossier en détail ls -l chemin_dossier
Lister le contenu d'un dossier, y compris les dossiers cachés ls -a chemin_dossier
Lister le contenu d'un dossier avec la taille des fichiers ls -h chemin_dossier
Trier les fichiers par date, du plus récent au plus ancien ls -t chemin_dossier
Trier les fichiers par taille décroissante ls -S chemin_dossier
Inverser le tri ls -r chemin_dossier
Compter le nombre de fichiers présents dans un dossier ls -al | wc -l
Compter le nombre de fichiers présents dans un dossier, sans compter "." et ".." ls -1 | wc -l


LDD : Lister les dépendances
Lister les dépendances d'un exécutable ldd -v nom_executable


HEAD : Voir le début d'un fichier
Voir les premières lignes d'un fichier head fichier.txt
Supprimer les (n) premières lignes d'un fichier (en gardant un fichier bak) head -(n) fichier.txt > nouveau_fichier.txt


TAIL : Voir la fin d'un fichier
Voir les dernières lignes d'un fichier tail fichier.txt


DATE : Manipuler des dates
Afficher la date date
Afficher la date au format temps universelle UTC date -u
Afficher la date sous forme de timestamp (nombres de secondes écoulées depuis le 01/01/1970 à 00:00:00) date +%s
Afficher la date d'une façon structurée date -u +"%Y-%m-%d %H:%M:%S UTC"
Afficher une date spécifique sous forme de timestamp date -d "yyyy-mm-dd HH:MM:SS" +%s
Convertir un timestamp en date date -d @timestamp
Convertir un timestamp en date sous le format "YYYY-MM-DD HH:mm:ss" date -d @timestamp '+%Y-%m-%d %H:%M:%S'


TOP : Afficher les processus
Affiche les processus actuellement en cours top


FREE : Afficher la mémoire
Affiche la mémoire disponible free


PASTE : Copier un fichier dans un autre
Créer un fichier contenant les colonnes d'un fichier suivi des colonnes d'un second paste fichier_1.txt fichier_2.txt >> fichier_3.txt


SORT : Trier un fichier suivant une clé
Trier un fichier texte suivant une clé (par rapport à une colonne repérée par son numéro : 1, 2, etc ...) sort -k(clé) fichier.txt


SED : Substituer un mot par un autre
Substituer un mot par un autre dans un fichier sed -i -r 's/mot_a_substituer/nouveau_mot/g' nom_du_fichier
Supprimer les (n) dernieres lignes d'un fichier (en gardant un fichier bak) sed -i.bak "1,(n)d" fichier.txt
Supprimer toutes lignes d'un fichier contenant le mot "(mot)" sed -i '/(mot)/d' fichier.txt


GREP : capturer une chaine de caractères
Afficher les lignes (et le numéro de ligne) contenant le mot "(mot)" dans un fichier grep -n (mot) fichier.txt
Compter le nombre de lignes contenant le mot "(mot)" dans un fichier grep -n (mot) fichier.txt | wc -l


KILL : Tuer un processus
Tuer un processus kill pid_processus
Tuer violemment un processus (forcer) kill 9 pid_processus


Opérateur $ : Interpréter une commande
Interpréter une commande $( commande )
Interpréter un calcul arithmétique $(( calcul ))
Prendre une sous-chaine d'une chaine contenue dans une variable ${variable:debut:nombre_de_caractères}


GNOME-SCREENSHOT : Faire une capture d'écran avec le bureau Gnome
Faire une capture d'écran gnome-screenshot
Prendre une capture de la fenêtre qui a le focus gnome-screenshot --window
Prendre une capture de la fenêtre ou de l'écran, après le délai renseigné en secondes gnome-screenshot --delay=10
Prendre une capture qui comprend les bords de l'écran gnome-screenshot --include-border
Prendre une capture et ajouter un effet ombré tout autour gnome-screenshot --border-effect=shadow
Prendre une capture de la fenêtre ou de l'écran, et entourer la capture d'un cadre gnome-screenshot --border-effect=border
Prendre une capture de la zone que vous délimiterez avec la souris gnome-screenshot --area


GIT : versionner du code
Versionner un dossier avec GIT git init
Verifier l'etat d'un dossier versionne avec GIT et de ses fichiers git status
Ajouter un fichier git add
Commiter un fichier git commit -m "commentaire"
Indexer et commiter un fichier a la suite (raccourci) git commit -am "commentaire"
Consulter l'historique des commit git log
Récupérer un projet sur GitHub git clone adresse_du_repository
Envoyer fichier sur GitHub (apres commit) git push origin master
Recuperer un fichier depuis GitHub git pull origin master
Verifier les branches presentes git branch
Creer une branche nommee nom_branche git branch nom_branche
Se placer sur une branche git checkout nom_branche
Fusionner une branche nommee nom_branche avec la branche actuelle (sur laquelle on se trouve) git merge nom_branche
Mettre de cote une modification (les fichiers ne prendront donc pas en compte la derniere modification en cours) git stash
Prendre en compte la modification en cours (mise de cote jusqu'a present) git stash pop


NCDUMP : Visualiser le contenu d'un fichier NetCDF
Visualiser les variables ncdump -h fichier.nc
Visualiser la valeur d'une variable ncdump fichier.nc -v variable


AWK : Manipuler un fichier par colonne
Afficher la colonne n° X d'un fichier texte ayant comme séparateur de champ FS (par défaut : " ") awk -F "FS" '{ print $X }' fichier.txt


CHMOD : Changer les droits sur un fichier
Ajouter des droits à une entité chmod entité+droits fichier
Enlever des droits à une entité chmod entité-droits fichier
Ajouter (enlever) des droits à une entité d'une façon récursive sur les tous les fichiers du dossier chmod -R entité+(-)droits dossier
a (all) Toutes les entités
u (user) Le propriétaire du fichier
g (group) Le groupe auquel appartient le fichier
o (others) Les autres personnes (autres que le propriétaire et le groupe)
r (reading) Droit de lecture
w (writing) Droit d'écriture
x (execute) Droit d'exécution
Affecter des droits aux entités u, g et o chmod ugo (u, g et o sont des chiffres compris entre 0 et 6)
Aucun droit 0
Droit d'exécution seulement 1
Droit d'écriture seulement 2
Droit d'écriture et d'exécution 3
Droit de lecture seulement 4
Droit de lecture et d'exécution 3
Droit de lecture et d'écriture 6
Tous les droits 7
